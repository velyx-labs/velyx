## [1.0.0] - 2026-05-17

### Added
- Landing page improvements: enhanced hero with better badge, H1, and value proposition
- Feature cards with improved descriptions, larger icons, and "Learn more" links
- "What You Actually Get" section with reformulated feature highlights
- CTA banner with better copy and dual button actions
- Full Livewire Volt migration for all documentation pages
- Missing Livewire components for docs layout (Header, Footer)
- SPA navigation with `wire:navigate` throughout docs and homepage
- Enhanced docs/installation page with comprehensive setup guide (requirements, steps, next steps)
- Semantic versioning documentation (RELEASES.md)
- GitHub repository configuration:
  - Updated repo description and topic tags
  - Configured GitHub issue templates (already existed: bug, feature, docs, component)
  - v1.0.0 release tag
- Release workflow documentation for future versions

### Changed
- Landing page layout with improved typography, spacing, and visual hierarchy
- Feature card descriptions for better clarity and persuasiveness
- CTA copy across all sections
- Docs installation page expanded with detailed requirements and examples

### Fixed
- Missing Livewire docs header and footer components
- Broken wire:navigate links across docs pages
